Time to event, and whether the gap holds

Kaplan–Meier curves show how survival diverges between arms; the log-rank tests the difference and Cox quantifies it as a hazard ratio. But a single HR only means something if the hazards stay proportional — so the Schoenfeld check comes built in.

How to read this · limitations. A single hazard ratio assumes proportional hazards; when the Schoenfeld test rejects PH, the HR is a misleading time-average and interval HRs or RMST are the honest summary. The log-rank test shares the PH sensitivity. Demo data are illustrative.
