Comparing several treatments at once

When trials don't all test the same pair, a network borrows strength across direct and indirect comparisons — but only if it is connected and consistent, and only if ranking respects the credible intervals.

Type 2 · Network MA — living · live re-run illustrative demo data

What this means — in plain language

When treatments have never been compared head-to-head, this combines direct and indirect evidence across a whole network to rank them — provided the indirect routes agree with the direct ones.

Estimation

Random-effects inverse-variance network meta-analysis (consistency model), reference = —. Between-study variance τ² estimated by a network method-of-moments (DerSimonian–Laird style); rankings by Monte-Carlo SUCRA with a fixed seed (deterministic).

Network meta-regression — does the modifier change effects?

—

Common-coefficient model (Cooper 2009 / NICE TSD3): one shared interaction term for the effect modifier across all active-vs-reference contrasts, fit by GLS at the network τ². It complements the transitivity check above — transitivity asks whether the modifier is balanced; this asks whether it matters.

06

Ranking — with credible intervals

never bare ranks

SUCRA alone misleads — read the intervals. Each row is a treatment's OR vs reference with its 95% credible interval; overlapping intervals mean a higher SUCRA does not prove superiority.

Rankogram — full rank-probability distribution

Each treatment's probability of holding every possible rank, across 3000 Monte-Carlo draws from the joint posterior — the distribution that SUCRA collapses into a single number. P(best) is the chance of being top-ranked; a high SUCRA with a spread-out rankogram signals real rank uncertainty.

07

League table

every pairwise contrast

Cell = odds ratio of the row treatment vs the column (<1 favours the row), with 95% credible interval.

Contribution matrix — where each estimate's evidence comes from

Each network relative effect is a weighted combination of the direct comparisons; cells show the % each direct comparison contributes to that estimate (hat-matrix / absolute-weight contributions, Krahn 2013). Rows sum to 100%. A network estimate dominated by one direct comparison inherits that comparison's risk of bias — read it alongside the league table.

—

    
How to read this · limitations. A network meta-analysis is valid only if the network is connected and consistent. This capsule tests consistency directly — node-splitting (SIDE) on every comparison with both direct and indirect evidence, and the global design-by-treatment Qinc (Jackson decomposition); when they disagree, the network estimates and ranking are unreliable. Node-splitting needs closed loops, and both tests have low power with few studies, so a non-significant result is not proof of consistency. Never rank on SUCRA alone — read the credible intervals and the rankogram. Indirect comparisons also assume transitivity (effect modifiers balanced across comparisons) — the transitivity panel is a descriptive check on one modifier, not a formal test. Demo data are illustrative.
