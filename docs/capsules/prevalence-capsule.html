How common is it?

A single proportion pooled across studies — on the logit scale to respect the 0–1 boundary — with a prediction interval that says where the next setting's prevalence would plausibly fall.

Type 4 · Proportion — living · live re-run illustrative demo data

What this means — in plain language

This pools how common something is across studies, on a scale that respects the 0–100% boundary, and gives a range for where the next setting's rate would plausibly fall.

How to read this · limitations. Prevalence is almost always heterogeneous across settings — read the prediction interval, not the confidence interval, as the honest summary for a new population. With few studies τ² (and so the interval) is imprecise. A 0.5 continuity correction is applied only to studies with 0 or N events. Demo data is illustrative.
