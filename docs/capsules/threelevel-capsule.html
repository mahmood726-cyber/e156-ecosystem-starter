When effect sizes come in clusters

Several effect sizes from the same study, lab or region are not independent — they share whatever makes that cluster distinct. A three-level model splits the heterogeneity into variation between clusters and variation within them, on top of pure sampling error, so the pooled effect isn't fooled by clusters that contribute many correlated estimates.

What this means — in plain language

When several effect sizes come from the same study or lab they aren't independent. This splits the variation into within- and between-cluster parts, so the pooled effect isn't fooled by clusters that contribute many similar estimates.

Three sources of variation

σ² by level

The nested model

Level 1Sampling variance — each effect size's own standard error squared (known). Level 2σ²₂ : variation between effect sizes within a cluster (e.g. several outcomes in one study). Level 3σ²₃ : variation between clusters (study, lab, region). EstimandThe overall mean effect, with standard error that accounts for both random levels via GLS.

Variance components are estimated by REML (a 2-D search over σ²₂, σ²₃ ≥ 0 maximising the restricted likelihood). Ignoring the nesting (a two-level model) understates the standard error when clusters are unbalanced — compared side by side in stage 5.

05

Three-level vs two-level

—

A standard two-level random-effects model treats all effect sizes as independent, so duplicated estimates from one big cluster get counted as if they were separate studies. The three-level model down-weights them — usually widening the CI. The gap is the cost of ignoring the nesting.
