What a positive result actually means

Sensitivity and specificity are properties of a test; predictive values are not — they swing with how common the disease is. A great test can still yield mostly false positives in a low-prevalence setting.

Primary · STARD — living · live re-run illustrative demo data

What this means — in plain language

From a single test's 2×2 results this computes its sensitivity, specificity and predictive values — and shows how the predictive values swing depending on how common the disease is.

How to read this · limitations. Sensitivity and specificity are properties of the test; predictive values (PPV/NPV) depend on disease prevalence in the tested population, so they do not transport to a setting with different prevalence. Needs a proper reference standard applied to all. Demo counts are illustrative.
