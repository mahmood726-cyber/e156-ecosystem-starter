How many patients is this result?

A "significant" trial can rest on a handful of outcomes. The fragility index counts the smallest number of patients whose outcome would have to flip to erase the significance — and compares it to how many were lost to follow-up.

Meta-research · fragility — living · live re-run illustrative demo data

What this means — in plain language

How sturdy is a 'significant' trial? This counts how few patients would need a different outcome to erase the result — a small number means the finding hangs by a thread.

01

Protocol

Question

InputA two-arm trial with a binary outcome, reported as a 2×2 table. TestFisher's exact test (two-sided) — the standard for the fragility index. EstimandFragility index: minimum events to add to one arm to push p to ≥ 0.05; fragility quotient = FI / total N. RuleModify one arm only (the arm with fewer events), one patient at a time.

Source

03

Eligibility

The fragility index is defined for a 2-arm, 1:1-ish parallel trial with a dichotomous outcome and an initially significant result (Fisher p < 0.05). For a non-significant result the capsule reports the reverse fragility index instead.

06

Interpretation

—

A small fragility index — especially one smaller than the number lost to follow-up — means the result could be reversed by reclassifying a few patients, so it should be read cautiously regardless of the p-value. The fragility index is descriptive, not a hypothesis test; it does not replace the confidence interval, and it depends on the (Fisher's exact, one-arm) convention used here.

Assurance

    
Same self-auditing trust layer: significance is judged by Fisher's exact test (not a normal approximation), exactly one arm is modified one patient at a time (the documented convention), and the index is shown beside the number lost to follow-up so a fragile result cannot hide behind its p-value. How to read this · limitations. The fragility index counts how many single-patient outcome flips erase significance — it modifies one arm only and is meaningful for a 2×2 trial whose result was significant; a small index does not by itself prove the trial is wrong. Demo counts are illustrative.
