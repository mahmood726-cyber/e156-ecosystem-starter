How sure are we it's worth it?

A point ICER hides uncertainty. Probabilistic sensitivity analysis samples the cost and effect inputs thousands of times; the acceptability curve then reads off the probability the intervention is cost-effective at every willingness-to-pay.

HTA · CEAC — living · live re-run illustrative demo data

What this means — in plain language

Because costs and benefits are uncertain, this shows the probability a treatment is good value at every willingness-to-pay — so you can see how confident the 'cost-effective' verdict actually is.

Protocol

Question

MethodProbabilistic sensitivity analysis: sample (ΔCost, ΔQALY) from their joint distribution. CEACAt each λ, the probability that incremental net monetary benefit (λ·ΔQALY − ΔCost) > 0. EstimandProbability the intervention is cost-effective, as a function of λ. ReproducibilityA seeded PRNG — the same inputs always give the same curve.
02

Distributions

Distribution parameters are illustrative and editable in stage 4; the scatter and CEAC re-run as you type. ΔCost and ΔQALY are sampled as correlated normals (a simplification — real PSA uses gamma/beta on the parameters).

03

Sampling

Samples are drawn with a deterministic seeded generator, so the curve is exactly reproducible — no two viewers see a different answer. Increase the sample count for a smoother CEAC.

How to read this · limitations. A cost-effectiveness acceptability curve shows the probability each option is cost-effective across willingness-to-pay thresholds; it depends entirely on the PSA input distributions and does not by itself identify the optimal decision (that is the strategy with the highest expected net benefit). Demo data are illustrative.
