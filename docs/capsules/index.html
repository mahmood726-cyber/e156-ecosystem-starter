Self-auditing, fully offline, editable single-file analysis tools. Each one carries its own data, runs its own statistics in the browser, and checks itself with a live self-audit before asking you to trust it.

How to use. Click any capsule — it opens as a single web page. Edit the example data in the table and every estimate, chart and self-audit check re-runs instantly. The assurance ribbon at the top shows whether the capsule currently agrees with itself. Everything runs locally in your browser; nothing is uploaded.
