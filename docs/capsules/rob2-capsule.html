e156 living evidence capsule · risk of bias (RoB2)

The overall judgement should follow from the domains

Cochrane's RoB2 turns five domain judgements into one overall verdict by a fixed algorithm — not reviewer gestalt. This capsule applies that algorithm live, so the overall column can never silently disagree with the domains feeding it.

Type 9 · RoB2 — living · live re-run illustrative demo data

What this means — in plain language

This walks each trial through the Cochrane risk-of-bias domains and derives one overall judgement by the algorithm, so the rating follows the answers rather than a gut feeling.

The RoB2 overall algorithm

Applied to every study, in order:

  1. High — if any domain is High.
  2. Some concerns — else if any domain raises some concerns.
  3. Low — else (all five domains Low).

—

RoB2 also allows escalating a study with multiple some-concerns domains to High when that "substantially lowers confidence". This capsule keeps such studies at Some concerns automatically but flags them (≥3 domains) so a reviewer can make that call deliberately — it is a judgement, not arithmetic.

Assurance

    
Same self-auditing trust layer: the overall judgement is derived from the domains by the published RoB2 algorithm and cannot be hand-set to contradict them, the algorithm itself is printed beside the result, and the discretionary "multiple concerns → High" escalation is surfaced rather than silently applied.
