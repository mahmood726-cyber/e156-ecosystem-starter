e156 living evidence capsule · GRADE Summary of Findings

How sure are we, and what does it mean in people?

GRADE turns eight domain judgements into one certainty rating, and a relative effect into an absolute one — events per 1000. This capsule derives both live, so the certainty can never drift from the domains and the absolute effect always matches the relative one.

Type 12 · GRADE SoF — living · live re-run illustrative demo data

What this means — in plain language

This turns the evidence into one plain certainty rating and an absolute answer — how many people per 1,000 are helped — so the conclusion can never drift from the judgements and numbers underneath it.

06

Certainty algorithm

How each rating was reached

The GRADE algorithm

Start: RCT → High (4 pips), observational → Low (2). Then:

  • − 1 per serious, − 2 per very serious concern across the five downgrade domains.
  • + 1 or + 2 for a large effect, + 1 for a dose-response gradient, + 1 if plausible confounding would only reduce the effect (observational bodies).
  • Clamp to High (4) · Moderate (3) · Low (2) · Very low (≤1).

08
Assurance

    
Same self-auditing trust layer: certainty is derived from the eight GRADE domains by the published algorithm (and cannot be hand-set to contradict them), each downgrade is itemised so a reader can audit the reasoning, and the absolute effect is always computed from the same relative effect shown beside it. Accent is seeded from this capsule's slug. Tier reaches Gold only on independent reproduction.

    
Self-audit — internal-consistency checks

      
Every check runs live against the current data. The capsule must agree with itself before asking the world to agree with it.
