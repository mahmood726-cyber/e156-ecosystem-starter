Significant on average, unreliable in the next setting

A confidence interval describes the average effect; a prediction interval describes the effect a new study would see. When the CI clears the null but the PI does not, "significant" hides a literature that could go either way next time.

Meta-research · PI gap — living · live re-run illustrative demo data

What this means — in plain language

An average effect can be statistically significant yet unreliable for the next setting. This flags when the pooled interval excludes 'no effect' but the prediction interval — where a new study would land — does not.

06

Interpretation

—

The prediction interval should be reported alongside the CI whenever heterogeneity is non-trivial; a CI that excludes the null with a PI that includes it means the pooled "effect" may not hold in a new population. The PI here uses the t-distribution with k−1 degrees of freedom (Cochrane Handbook v6.5) and is undefined for k < 2.
