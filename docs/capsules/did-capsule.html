What the trend would have been

A policy hits one group and not another. Difference-in-differences subtracts the control group's before-after change from the treated group's — using the control as the counterfactual trend the treated group would have followed anyway.

Causal · DiD — living · live re-run illustrative demo data

What this means — in plain language

This estimates a policy's effect by comparing the change in a treated group with the change in an untreated one, assuming both would otherwise have moved in parallel. The dashed line is that assumed 'what would have happened'.

Assumption

The parallel-trends assumption is untestable with two periods alone. With ≥2 pre-periods, an event study checks whether the groups tracked together before the intervention; this capsule uses the canonical 2×2 and flags the assumption explicitly.

06

Interpretation

—

The DiD effect is the vertical gap between the treated group's observed post value and its counterfactual (pre value plus the control group's change). It is only causal if parallel trends held — verify with pre-period data, not just this 2×2. Standard errors here assume independent group-period means; clustered/serial-correlation-robust SEs are usually larger in practice.

How to read this · limitations. Difference-in-differences rests on the parallel-trends assumption: absent the intervention, treated and control groups would have moved together. It cannot be proven, only supported by matching pre-period trends — inspect them before trusting the estimate. Demo data are illustrative.
