Does more change the risk?

A two-stage dose-response meta-analysis: a log-linear trend per study, pooled with random effects, drawn as a curve whose uncertainty band widens as you extrapolate.

Type 5 · Dose-response — living · live re-run illustrative demo data

What this means — in plain language

This traces how the effect changes across doses, pooling the trials at each level, so you can see the shape of the relationship and where the uncertainty widens.

Protocol

Search

Registry + database search, de-duplicated. Yields illustrative; dose-level data are editable in stage 4.

03

Screening

Two-reviewer screening to studies reporting ≥2 dose categories with an RR vs a common reference dose.

04

Extraction — editable dose levels

every row traces to a study

One row per dose category: study, dose, RR vs the reference dose, 95% CI. The lowest dose in each study is its reference (RR≈1). Edit, add or remove rows — the curve, pooled slope and trend test re-run live.

Trend & heterogeneity

—

A significant pooled slope is evidence of a linear trend; high between-study heterogeneity in slopes, or curvature in the points, signals nonlinearity (fit restricted cubic splines in dosresmeta).
