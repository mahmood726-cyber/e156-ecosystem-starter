Does this literature have evidential value?

If a set of significant findings reflects a real effect, their p-values pile up near zero — a right-skewed p-curve. If they were p-hacked or reflect no effect, the curve flattens. p-curve reads the shape of significance itself.

Meta-research · p-curve — living · live re-run illustrative demo data

What this means — in plain language

If a batch of significant findings reflects a real effect, their p-values pile up just under 0.05 in a tell-tale right-skew. This tests for that shape — and for the flat pattern that signals nothing underneath.

06

Verdict

—

Inference rules (Simonsohn, Nelson & Simmons): evidential value present if the half p-curve right-skew test is p<.05, or both right-skew tests are p<.1. Evidential value inadequate if the 33%-power flatness test is p<.05 (full or half). Otherwise inconclusive. p-curve cannot detect effects from non-significant studies, and is fooled if the wrong test is selected.

08
Assurance

    
Same self-auditing trust layer: only p<.05 enter the curve (enforced, not assumed), the half p-curve (p<.025) is reported alongside the full one to guard against ambitious p-hacking, and both the evidential-value and the flatness tests are shown so a flat curve can never be spun as support. Fisher's method combines pp-values exactly (χ² with 2k df). Accent is seeded from this capsule's slug. Tier reaches Gold only on independent reproduction.

Every check runs live against the current data. The capsule must agree with itself before asking the world to agree with it.

How to read this · limitations. P-curve assumes the analysed tests are statistically independent and were each selected against an a-priori prediction; it diagnoses evidential value but cannot rescue a biased set of studies. Demo p-values are illustrative.
