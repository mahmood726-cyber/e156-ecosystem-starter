A probability, not just an interval

A Bayesian random-effects model returns the full posterior of the pooled effect — so it can answer the question clinicians actually ask: how probable is benefit? It also borrows strength across trials, shrinking small noisy studies toward the consensus.

Type 7 · Bayesian — living · live re-run illustrative demo data

What this means — in plain language

Instead of one pooled number, this shows the whole range of effects that are plausible given the trials and your prior beliefs — and the probability the treatment truly helps. The wider the posterior, the less settled the evidence.

Prior sensitivity

τ is given a weakly-informative half-normal prior — recommended for the heterogeneity SD with few trials, where a flat prior over-states τ. Change the scale and watch the posterior move: a result that swings wildly is prior-driven, not data-driven.

—

—

The posterior is computed exactly on a grid over (μ, τ): for the normal–normal model each trial's contribution N(yᵢ | μ, sᵢ²+τ²) is the analytic marginal after integrating out θᵢ, so no MCMC is needed. Grid approximation is appropriate for this 2-parameter unimodal posterior; the dot above flags if posterior mass reaches a grid edge (widen the range). For ≥3 parameters, use the MCMC script below.

08
Assurance

    
Same self-auditing trust layer: results are reported as posterior credible intervals (not confidence intervals), the heterogeneity prior is stated and is adjustable for sensitivity, and a grid-edge diagnostic stands in for the convergence check an MCMC run would need. Pooling is on the log scale and back-transformed. Accent is seeded from this capsule's slug. Tier reaches Gold only on independent reproduction.

How to read this · limitations. Results depend on the prior, especially for τ (between-study heterogeneity) when studies are few — vary the prior scale and watch P(benefit) move. The posterior is exact on a grid, valid here but not for many parameters. Demo data is illustrative.
