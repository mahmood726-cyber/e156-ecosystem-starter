E156 Ecosystem Starter

Sanidi kompyuta yako ya utafiti kama Mahmood Ahmad anavyofanya (kiongozi wa utafiti wa magonjwa ya moyo, Makerere na Ziauddin).

Kumbuka: huu ni utoleo fupi kwa Kiswahili. Toleo kamili la Kiingereza lina maelezo yote (mahitaji ya awali, utatuzi wa matatizo, muundo wa usalama, miradi ya mfano). Amri za usakinishaji hapa chini ni zilezile katika lugha zote. Tafsiri kamili ya Kiswahili inasubiri wachangiaji — saidia kuiboresha.

Rahisi zaidi — fungua kwenye kivinjari, hakuna terminal

Kama hujawahi kutumia terminal, bonyeza kitufe hiki. Nafasi ya kazi ya bure ya GitHub itafunguka ikiwa na mfumo mzima umesakinishwa tayari: kanuni, kumbukumbu, Sentinel, Overmind, na Gemini CLI. Hakuna kinachosakinishwa kwenye kompyuta yako.

Fungua katika GitHub Codespaces →

Ujenzi ~dakika 2-3. Bora kwenye kompyuta — Codespaces kwenye kivinjari cha simu hufanya kazi lakini terminal si thabiti.

Ukitumia Codespaces, umemaliza kusakinisha — ruka sehemu ya "Mahitaji ya awali". Python, R na Node vimesakinishwa tayari kwa ajili yako kwenye wingu.
Simamisha codespace ukimaliza. Codespace iliyowazi hutumia saa zako za kila mwezi hadi isimame yenyewe (kawaida: dakika 30). Kusimamisha mara moja: nenda github.com/codespaces kisha bonyeza ⋯ → Stop codespace. Codespace iliyosimama bado hutumia hifadhi yako ya bure (GB 15) — GitHub huacha kuhesabu tu unapoifuta (⋯ → Delete). Data kubwa inaweza kufikia kikomo cha hifadhi kabla ya saa zako kuisha, kwa hivyo futa codespace ulizomaliza.

Nini kipya

Zana zimeongezeka na bado ni za nje ya mtandao na rafiki kwa mpango wa bure: wasaidizi wa nje ya mtandao (reuse.py, recall.py) wanaozuia kutumia tokeni kutengeneza upya kile kilichopo tayari; kifaa cha kutoa data cha RCT sasa kinatambua maeneo 17 ya magonjwa (VVU, malaria, homa ya matumbo, sickle cell, kifua kikuu…); tabaka tano za zana za meta-uchambuzi za nje ya mtandao; na kikagua-hesabu cha Overmind (v1.0.0). Maelezo kamili katika sehemu ya "What's new" ya ukurasa wa Kiingereza.

Usakinishaji — Windows

Fungua PowerShell (si Command Prompt). Bonyeza Win+R, andika powershell, kisha Enter. Bandika:

irm https://mahmood726-cyber.github.io/e156-ecosystem-starter/bootstrap.ps1 | iex

Usakinishaji — macOS / Linux / WSL

curl -fsSL https://mahmood726-cyber.github.io/e156-ecosystem-starter/e156-setup.sh | bash

Njia salama zaidi (inayopendekezwa kwa wanaoanza): nakili hazina kwanza, soma msimbo, kisha uendeshe — ona Kiingereza: Install on your own machine.

Thibitisha usakinishaji

# Windows PowerShell
ls $env:USERPROFILE\.claude\rules

# macOS / Linux / WSL
ls ~/.claude/rules

Unapaswa kuona faili nne za .md: e156, lessons, advanced-stats, rules.

Mahitaji ya awali (kwa usakinishaji wa kompyuta yako tu)

  • Python 3.11+ kutoka python.org/downloads (weka alama "Add python.exe to PATH") — lazima.
  • Node.js 20+ kutoka nodejs.org — lazima.
  • R 4.5+ — si lazima; kinahitajika tu kwa ukaguzi wa ziada wa R (metafor). Karatasi yako ya kwanza ya maneno 156 haihitaji R.
  • CLI ya wakala wa AI: Gemini CLI (bure), Claude Code, au OpenAI Codex.
